# TeamSurvivalPlugin

Minecraft Paper/Spigot plugin koji dodaje timski survival sistem:
- Timovi sa pozivnicama
- Team Home
- Team chat
- TPA samo unutar tima
- Respawn na Team Home
- Combat tag (blokira teleport)
- Warmup + cooldown teleport sistem
- Locator teammate-ova (ActionBar)
- TAB i nametag boje po timu
- Persistencija u teams.yml
